# Rock, Paper, Scissors (Python)

A simple, interactive command-line implementation of the classic Rock, Paper, Scissors game built using Python. Play against the computer and see who wins!

## Features
* **User vs. Computer:** Play against an automated opponent utilizing random selections.
* **Input Validation:** Handles invalid user inputs gracefully without crashing.

## Prerequisites
To run this game, you only need Python installed on your system:
* Python 3.x
